# AnonymousGroupXBot
Telegram-бот для анонимного общения со случайными людьми на Python.


Функционал:

1. Функция анонимного общения в случайной группе из 10 человек.
2. Анонимный чат 1 на 1 со случайным собеседником.
3. Функция настройки поиска собеседников по трём критериям (пол, страна и хобби).
4. Функция выбора ника в анонимном чате.
5. Функция удаления своих сообщений у собеседников.
6. Функция просмотра данных собеседников (пол, страна и хобби).
7. История собеседников с функцией возврата.
8. Функция просмотра своей статистики и собеседников.
9. Функция добавки в друзья.
10. Топ пользователей.


Библиотеки:

1. Aiogram;
2. Sqlite3;
3. YooMoney (для приёма платежей);
4. Aiohttp.
